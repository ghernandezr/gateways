# Gateways project

## How to setup

### Cloning and install dependencies

```bash
git clone https://github.com/ghernandezr/gateways.git

cd gateways

npm install
```

### Run in dev mode

```bash
npm run dev
```

### Run in dev mode with nodemon

```bash
npm start
```

### Run in prod mode

```bash
npm run prod
```

### Execute test

```bash
npm run test
```
